# Matlosen Web

Landing page and legal pages for [matlosen.no](https://matlosen.no).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed automatically to Vercel on push to `main`.

## Pages

- `/` - Landing page
- `/personvern` - Privacy Policy (Personvernerklæring)
- `/vilkar` - Terms of Use (Bruksvilkår)

## Adding Screenshots

Replace the placeholder images in the phone mockups by:

1. Take screenshots from the app
2. Add them to `public/screenshots/`
3. Update `app/page.tsx` to use the real images
